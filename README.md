## Introduction üìäüìà

This project was developed as part of the 25/2 assessment for a Applied Statistics.  
The goal of the assessment is to explore key concepts of **statistical inference** using **simulation**, implemented in **Python** with Jupyter Notebook.

Throughout the project, I use computational simulations to better understand theoretical ideas such as:

- Randomisation and probability
- Sampling distributions
- Type I and Type II errors
- Statistical power
- t-tests and ANOVA

Instead of relying only on formulas, simulations are used to illustrate how these concepts behave in practice.

All problems are implemented in a single notebook: **`problems.ipynb`**, following the instructions provided in `problems.md`.

## Problems Overview üß™

### Problem 1 ‚Äî Lady Tasting Tea
Simulation of Fisher‚Äôs classic experiment and an extended version with more cups, comparing exact and simulated probabilities.

### Problem 2 ‚Äî Normal Distribution
Comparison between sample and population standard deviation using repeated sampling from a normal distribution.

### Problem 3 ‚Äî t-Tests and Type II Error
Simulation study showing how the type II error rate and statistical power change as the true difference in means increases.

### Problem 4 ‚Äî ANOVA
Comparison between one-way ANOVA and multiple pairwise t-tests, with a discussion of why ANOVA is preferred.
